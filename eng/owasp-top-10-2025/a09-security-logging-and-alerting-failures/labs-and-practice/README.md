## Scope

The completed work focused on application-level AppSec review rather than SOC or SIEM administration.

The exercises covered:

- identifying authoritative security-event sources,
- defining structured authentication events,
- distinguishing events from detections and alerts,
- proposing basic correlation rules,
- reviewing a string-built log entry for injection risk,
- classifying audit fields and secrets,
- writing developer-focused remediation and regression tests.

No dedicated PortSwigger lab was forced because the completed A09 work was better represented by theory, code review, detection design, and controlled application examples.
