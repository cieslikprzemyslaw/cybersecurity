This section collects practical container security notes for Docker-based application work.

| Module | Scope |
|---|---|

## Current Focus

The current module focuses on a local Node.js / Express API, React / Vite frontend, Prisma, SQLite, uploads, nginx and Docker Compose stack.

These notes are defensive and developer-oriented: they connect container build/runtime decisions with AppSec review, safer deployment habits and regression-friendly validation.
