# XSS Labs — Summary Index

These notes summarise legal training labs completed as part of the XSS learning topic.

They are not full walkthroughs or exploit guides. The focus is on:

- what was tested,
- what the root cause was,
- what the impact could be,
- how developers should fix the issue,
- what I learned as a Frontend Engineer.
