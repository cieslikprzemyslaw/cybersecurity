# XSS Labs — indeks podsumowań

Te notatki podsumowują legalne laby wykonane w ramach tematu XSS.

To nie są pełne walkthroughy ani exploit guides. Skupiam się na:

- co testowałem,
- jaki był root cause,
- jaki mógł być impact,
- jak developer powinien to naprawić,
- czego nauczyłem się jako Frontend Engineer.
