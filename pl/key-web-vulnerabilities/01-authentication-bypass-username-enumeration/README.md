## Zakres tematu

Ten moduł obejmuje:

- authentication versus authorization,
- wzorce authentication bypass,
- sygnały username enumeration,
- porównywanie odpowiedzi logowania,
- zachowanie account lockout,
- odpowiedzialność frontendu i backendu,
- remediation z perspektywy developera.
