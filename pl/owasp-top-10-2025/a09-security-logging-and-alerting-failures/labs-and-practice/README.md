## Zakres

Ukończona praca koncentrowała się na review AppSec na poziomie aplikacji, a nie na administracji SOC lub SIEM.

Ćwiczenia obejmowały:

- identyfikowanie autorytatywnych źródeł security eventów,
- definiowanie strukturalnych eventów authentication,
- rozróżnianie eventów, detection i alertów,
- proponowanie podstawowych reguł korelacji,
- review log entry zbudowanego ze stringów pod kątem injection,
- klasyfikację pól auditowych i sekretów,
- pisanie remediation i testów regresji z perspektywy developera.

Nie wymuszono osobnego labu PortSwigger, ponieważ ukończona praca A09 była lepiej reprezentowana przez teorię, code review, projektowanie detekcji i kontrolowane przykłady aplikacyjne.
