## Zakres

Praca koncentrowała się na application-level AppSec review, a nie na administracji SOC lub SIEM.

Ćwiczenia obejmowały:

- wskazanie autorytatywnych źródeł security events,
- projekt structured authentication events,
- rozdzielenie eventu, detekcji i alertu,
- podstawowe correlation rules,
- review ręcznie składanego log line pod kątem injection,
- klasyfikację audit fields i secrets,
- developer-focused remediation i regression tests.

Nie wymuszałem PortSwigger labu, ponieważ A09 lepiej pokryły teoria, code review, detection design i kontrolowane przykłady aplikacyjne.
