# STRIDE Cheat Sheet

- **Spoofing** - podszywanie sie pod innego uzytkownika, usluge albo system
- **Tampering** - modyfikowanie danych, requestow, komunikatow albo zapisanych informacji
- **Repudiation** - zaprzeczanie wykonaniu akcji, gdy system nie ma wiarygodnych dowodow audytowych
- **Information Disclosure** - ujawnienie danych nieuprawnionej stronie
- **Denial of Service** - uniemozliwienie korzystania z systemu albo funkcji
- **Elevation of Privilege** - uzyskanie uprawnien wiekszych niz pierwotnie przyznane

Uzywaj STRIDE jako checklisty dla kazdego aktora, procesu, magazynu danych, przeplywu danych i granicy zaufania. Wpis STRIDE oznacza potencjalne zagrozenie, a nie dowod istnienia podatnosci.
